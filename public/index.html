Turnipinator
Javascript Typescript Node.js Discord

Turnipinator is a discord bot made for the Dutch Super Smash Brothers Discord. It handles user role requests via a web interface and streams Youtube audio to voice channels.
The bot also provides a quick and easy way for tournament organizers to add their Smash tournaments to the Tournament agenda channel.

Dutch Smash Agenda
Javascript PWA Lit-HTML

The Dutch Smash Agenda is a progressive web app that features a live list of up-and-coming Super Smash Brothers tournaments in the Netherlands. This agenda is the main source competitors use to find locals and national tournaments. Being a part of the Dutch Smash Discord, Tournament organizers can add their own tournaments via a web portal. The agenda also features live updating information like the amount of sign ups.
